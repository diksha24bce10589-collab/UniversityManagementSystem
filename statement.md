# Problem Statement
Managing university operations manually is time-consuming, error‑prone, and inefficient. This project aims to automate academic and administrative workflows involving students, teachers, examinations, and fee processing.

# Scope of the Project
- Handle student and teacher data  
- Manage course, attendance, and examination modules  
- Automate fee structure and payment handling  
- Provide a user‑friendly GUI for interactions  
- Maintain records securely within the system  

# Target Users
- University administrators  
- Teaching staff  
- Students  
- Accounts/fee department  

# High-Level Features
- Complete student & teacher management  
- GUI login and dashboard  
- Fee structure and payment module  
- Leave management system  
- Examination management module
