# University Management System

## Overview
This project is a Java-based University Management System designed to manage student, teacher, course, fee, and examination details. It includes a GUI interface built using Java Swing and organizes administrative functionalities in a structured manner.

## Features
- Student registration and details management  
- Teacher registration and details management  
- Course and examination management  
- Fee structure and payment module  
- Leave management for students and teachers  
- GUI-based login and dashboard  
- Modular Java classes for each feature  

## Technologies / Tools Used
- **Java SE**  
- **Java Swing** (GUI)  
- **NetBeans IDE** (project structure & build)  
- **XML** (build configuration)
